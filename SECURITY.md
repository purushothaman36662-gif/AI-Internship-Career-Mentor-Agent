## Reporting a Vulnerability

As a project focused on agentic security perimeter boundaries, proactive identification of bypass vulnerabilities is highly valued. 

If you discover a method to bypass the input validation firewall middleware or execute an un-intercepted prompt injection attack, please report it by opening an issue on this repository or reaching out directly via email at purushothaman36662@gmail.com.
